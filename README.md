## ⚡ Как пользоваться на чемпионате

```
1. Утром идёшь в интернет-кафе (15 мин)
2. Открываешь листок нужного дня (ссылка выше)
3. Читаешь и запоминаешь / переписываешь ТОЛЬКО формулы
4. Работаешь с листком весь день
5. Вечером — следующий день готов
```

## 🏆 Критерии оценки (не забыть!)

| # | Правило |
|---|---------|
| 1 | **PascalCase** для классов и методов; `_camelCase` для приватных полей |
| 2 | **Без русских букв** в именах классов, переменных, папок |
| 3 | **SOLID** — проверяется на каждом модуле |
| 4 | **После каждого модуля:** Build + Export Package |
| 5 | Весь UI только через **Canvas** |
| 6 | JSON файлы в **StreamingAssets** (редактируемы в билде) |
